# Action Figures - Negócio

## Visão Geral

Negócio de venda de action figures através de múltiplos canais digitais, operando com modelo de estoque próprio (compra e revenda).

## Estrutura do Projeto

Este projeto contém a documentação e planejamento do negócio, incluindo:

- **Plano de Responsabilidades**: Definição clara das atribuições de cada sócio
- **Processos**: Documentação de processos operacionais
- **Métricas**: Dashboards e acompanhamento de KPIs

## Informações do Negócio

- **Modelo**: Estoque próprio (compra e revenda)
- **Estrutura legal**: CNPJ/Sociedade existente
- **Investimento**: 50/50
- **Canais de venda**:
